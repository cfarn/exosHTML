Questions / réponses exercice 1 - HTML

Question 1
La déclaration DOCTYPE est-elle obligatoire ? Où doit-elle être placée ? Quelles informations contient-elle ?
Réponse 1 : En html, DOCTYPE est le préambule requis en haut de tous les documents. Elle se place en tête du code devant la balise <html> Elle contient le nom de l'élément racine.
Question 2
Que faut-il faire pour que le code HTML soit valide ?
Réponse 2 : Il faut respecter l'inclusion des éléments et les attributs propres à chacun d'entre eux et ensuite vérifier avec le validateur du W3C. Validateur du W3C
Question 3
À quel emplacement est defini le jeu de caractères utilisé dans le document ?
Réponse 3 : Le jeu de caractères est précisé dans un élément <meta> inclus dans <head> avec l'attribut charset inclus dans <head><meta charset="UTF-8">
Question 4
Où est déclaré le nom de l'élément racine du document ? Quel est son rôle et son nom ?
Réponse 4 : L'élément HTML représente la racine d'un document HTML. Il est déclaré dans DOCTYPE. C'est l'élément qui inclus tous les autres et tout autre élément du document doit être un descendant de cet élément.
Question 5
Quels sont les 2 éléments enfants de l'élément HTML ?
Réponse 5 : Les enfants (de 1er niveau) <head> et <body>.
Question 6
À quoi sert l'élement <head> ? À quoi sert-il ?
Réponse 6 : La balise HEAD contient les paramètres d'en-tête de la page html qui sont invisibles pour l'utilisateur (sauf le title) : il contient BASE, LINK, META, SCRIPT, STYLE, TITLE. Seuls les éléments BASE et TITLE ne doivent être utilisés qu'une seule fois.
Question 7
Quel élement est obligatoire dans l'élément <head> ? À quoi sert-il ?
Réponse 7 : Seul TITLE est obligatoire. C'est le titre dans l'onglet du navigateur.
Question 8
Écrire le code nécessaire à la liaison d'une feuille de style avec un document
Réponse 8 : <link rel="stylesheet" href="css/styles.css"
Question 9
Écrire le code nécessaire à la liaison d'un document externe contenant des scripts JavaScript avec un document
Réponse 9 : <script> src="javascript.js"<script>
Question 10
Comment déclarer les mots-clefs associés au site ? Quelle est l'utilité de cette déclaration ? Écrivez-un exemple
Réponse 10 : Dans la balise meta name content : <meta name="keywords" content="Exercices ebook, HTML, Cinnie Farnaud">. C'est utile pour le référencement naturel.
Question 11
Quel est le rôle de l'élément <base>
Réponse 11 : C'est l'URL de base à utiliser pour recomposer toutes les URL relatives contenues dans un document. Il ne peut y avoir qu'un seul élément BASE au sein d'un document.
Question 12
Peut-on écrire le code suivant dans la page ?
<body>
Bienvenue dans notre site <h1>Le site du jour</h1>
</body>
Réponse 12 : Non car l'élément body contient du texte directement alors qu'il devrait être inclus dans un des éléments enfants de BODY.
Question 13
Peut-on inclure les éléments <tbody>, <form> et <li> directement dans l'élément <body> ?
Réponse 13 : Non, TBODY est enfant de la balise TABLE et LI est enfant de UL ou OL. En revanche, FORM peut être inclus directement dans BODY car il ouvre un formulaire.
