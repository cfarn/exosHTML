Aide-mémoire sélecteurs css et leurs propriétés

Les sélecteurs css

  • * : sélectionne tous les éléments de la page
  • ma-jolie-class : sélectionne tous les éléments ayant le nom de class "ma ma-jolie-class"
  • mon-unique-id : sélectionne tous les éléments ayant pour id "mon-unique-id"
  • div, aside, article : sélectionne toutes les balises <div, <aside, <article
  • nav a : sélectionne tous les liens <a imbriqués dans la balise <nav
  • nav a:hover : sélectionne tous les liens de la nav et active une animation au survol de la souris

Propriétés css utilisées dans cette page

Décrivez leur usage et leurs valeurs possibles

  1. propriétés pour le texte
    • font-family : famille de polices dans un ordre de priorité
    • font-size : taille de la police
    • text-align : alignement horizontal du texte
    • line-height : interlignage
    • text-decoration : pour « décorer » le texte en ajoutant une ligne sous, sur ou à travers le texte pour le souligner
    • color : couleur du texte
  2. Propriétés pour les éléments
    • background : arrière-plan d'un élément
    • position : position d'un élément dans la page
    • height : hauteur d'un élément
    • width : largeur d'un élément par rapport à la taille de fenêtre si on indique une valeur proportionnelle en % ou en em
    • left : position à partir de la gauche de la fenêtre du navigateur
    • float : effet flottant. retirer un élément du flux normal et le placer sur le côté droit ou gauche de son contenu
    • margin : marges extérieures d'un élément
    • padding : marges intérieures, espacement entre l'élément et sa bordure
    • position : position d'un élément dans la page
    • top : position à partir du haut
    • transition : temps de transition lors d'une animation

Propriétés css utilisées dans cette page

Recherchez les propriétés ci-dessous, faites des tests avec sur cette page depuis la console de débuggage pour tester les valeurs qu'elles peuvent prendre

  • font-weight : épaisseur du texte
  • letter-spacing : espace entre les lettres
  • word-spacing : espace entre les mots
  • text-shadow : ombres sur le texte
  • background-image : image d'arrière-plan
  • background-size : taille d'arrière-plan
  • opacity : opacité de l'élément
  • border-style : style des lignes utilisées pour les bordures d'un élément
  • box-shadow : ombres sur la boîte d'un élément
  • list-style-image : images utilisées pour les puces dans une liste
  • border : propriétés liées à la bordure
  • border-radius : arrondir les coins
  • border-width : largeur de la bordure
  • font-variant : variations de police
  • background-color : couleur d'arrière-plan
  • !important ; : importer des règles à partir d'autres feuilles de style
